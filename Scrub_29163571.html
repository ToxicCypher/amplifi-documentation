Amplifi : Scrub

Created by Cameron Gibson, last modified on Jun 07, 2024

Summary

This option is only available to the Music and Ambience AmplifiContext type.

The Scrub enum within the Amplifi namespace defines different scrubbing modes that can be applied within the Amplifi system. Here's an explanation of each enum value:

Options

Option

Definition

None

The system will not modify or alter the current state or playback of audio content.

Random

The system will randomly select a section of audio content unpredictably when the specified node is triggered.

Half

The system will start at the halfway point of audio content when the specified node is triggered.
