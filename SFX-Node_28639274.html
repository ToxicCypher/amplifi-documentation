Amplifi : SFX Node

Created by Cameron Gibson, last modified on Jun 10, 2024

Summary

In this section, you will learn about all of the configuration that is available for SFX Amplifi Nodes and in general, how each SFX Amplifi Node subtype behaves.

SFX Node Subtype Behavior

Below is a table detailing all of the available SFX node subtypes and a detailed description regarding the general behavior of the SFX node subtypes.

Node Subtype

Description

Orphan

Sound effect(s) will be spawned at the location of this node.

Child of Other

Sound effect(s) will be spawned as a child of the game object that collided with this node.

Child of Node

Sound effect(s) will be spawned as a child of this node.

SFX Node Configuration

Below is a table detailing all of the available configuration for the SFX type Amplifi Node. If any SFX Amplifi Node subtypes are available, the availability of a given parameter for a particular subtype will be noted.

Parameter

Data Type

Sound Effects

List<AudioClip>

List<AudioClips> { }

Orphan | Child of Other | Child of Node

A list of AudioClips that can be played from this Node. If more than one clip is supplied, a clip is randomly selected. If a single clip is provided, that clip is guaranteed to play. If no clips are provided, this node will not instance any sound effects.

Context

AmplifiContext

null

Orphan | Child of Other | Child of Node

The AmplifiContext object that will drive the behavior of the AudioSources that are ultimately generated as a result of the target node.

Loop Sound Effect

bool

false

Orphan | Child of Other | Child of Node

If true, the SFX will play in a loop until the game object exits the nodes bounding volume. This property is automatically set to false and disabled when ‘Destroy When Triggered’ is true.

Destroy When Triggered

bool

false

Orphan | Child of Other | Child of Node

If true, the target node will be destroyed after the SFX is generated. This property is automatically set to false and disabled when ‘Loop Sound Effect’ is true.

Allowed Tags

List<string>

List<string> { }

Orphan | Child of Other | Child of Node

A list of tags that are allowed to trigger this node, given that the game object that enters the bounding volume is tagged with one of the tags supplied to this list. If no tags are specified, any game object can trigger the target node.
